# 丧尸文明：末日崛起

一款丧尸末日生存策略游戏，在崩溃的世界中建立避难所，管理幸存者，研究丧尸病毒，最终重建人类文明。

## 游戏特色

- 🧟 **丧尸末日** - 在丧尸横行的废墟世界中求生
- 🏗️ **基地建设** - 建造避难所，加固防线，保护幸存者
- 🔬 **病毒研究** - 研究丧尸病毒，开发疫苗
- ⚔️ **战斗系统** - 训练猎人，组建军队，清除丧尸
- 👥 **幸存者管理** - 管理不同角色的幸存者

## 技术栈

- React 19 + Vite
- Tailwind CSS
- Lucide React Icons

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
